Este projeto implementa um jogo de rolagem horizontal (side-\/scrolling) com terreno procedural para um LCD Nokia 5110 em um microcontrolador AVR.\hypertarget{index_features}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_features}

\begin{DoxyItemize}
\item Geração de terreno procedural usando tabelas de cossenos.
\item Controle de aceleração e desaceleração.
\item Desenho de bitmap do baja pixelado. 
\end{DoxyItemize}
